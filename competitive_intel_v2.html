Cost (SF bundle)

Threat Level

Unlimited responses, any object — W2L caps at 500 leads/day and only maps to the Lead object. FA handles any standard or custom object, any volume.

500 Lead/day hard limit — high-traffic campaigns will silently stop capturing leads mid-day. FA offers unlimited responses with reliable composite API sync.
Lead object only — cannot map to Contacts, Accounts, Cases, or any custom object. FA maps to any standard or custom object across all clouds.
No file uploads — a complete blocker for application forms, document collection, and regulated intake workflows.
